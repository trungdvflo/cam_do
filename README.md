This README would normally document whatever steps are necessary to get your application up and running.

### How do I get set up? ###

* Install NodeJS version 8.x

* Install Python 2.7

* Install bower:
```
	$ npm bower install -g
```

* Install gulp:
```
	$ npm gulp install -g
```

* Install nodemon:
```
	$ npm nodemon install -g
```
	
* Install tools:
	 if window:
```
	 $ npm install --global --production windows-build-tools
```
	 
* Install app:
```
	$ cd serverAPI
	$ npm install
	$ cd client
	$ npm install
	$ bower install
```
	
* Database configuration
	edit file: serverAPI/config/config.js
	
* Run app:
```
	$ cd serverAPI
	$ npm start
```
	http://localhost:3000
